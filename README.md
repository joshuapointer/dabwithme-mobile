# dabwith.me — Mobile App

The original Dab Rite companion app for iOS and Android, connecting your mobile device to the Dab Rite IR thermometer via Bluetooth Low Energy.

## What This Was

- **Project:** `dabwith.me`
- **Platform:** iOS & Android
- **Framework:** Expo SDK 53 (bare workflow)
- **React Native:** 0.83.6
- **Package Manager:** pnpm
- **Key Features:**
  - Bluetooth Low Energy connectivity to Dab Rite IR thermometer
  - Real-time temperature display
  - EAS (Expo Application Services) updates
  - React Native Reanimated for smooth UI
  - Custom BLE permissions for both iOS and Android

## Tech Stack

- **Expo SDK 53** — The core framework
- **React Native 0.83.6** — Cross-platform native UI
- **React Native Reanimated** — High-performance animations
- **React Native Gesture Handler** — Touch and gesture system
- **Expo Dev Client** — Development builds
- **Expo Updates** — Over-the-air updates via EAS
- **TypeScript** — Type safety

## Status

🚫 **Deprecated** — This was an early prototype of the Dab Rite companion app. The codebase has been superseded by [quartzos-app](https://github.com/joshuapointer/quartzos-app), which features:
- Skia-powered UI rendering
- Siri Bridge integration
- Expo SDK 54
- Enhanced BLE performance
- Cleaner architecture

## Historical Note

This repo represents the first iteration of connecting Dab Rite devices to mobile. It validated the core Bluetooth connectivity and established the foundation for what became the production-quality quartzos-app.
